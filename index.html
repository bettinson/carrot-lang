carrot-lang

This is Carrot lang. Carrot is a simple templating language that can be embedded into HTML. Right now, I'm working on using this for Imageup for custom user pages. Use file extension .html.crt and run preprocessor.rb to use it.

Usage

Embed carrot in html between {{ }}.

<body>
  {{ name="Matt" }}
  <p>Hi, This is {{ name }}'s website.</p>

  {{ photos }}
  <img src={{ photo.path }}>
  {{ photos }}
</body>

This will output the following html, eventually:

<body>
  <p>Hi, This is Matt's website.</p>

  <img src="photo1.jpg", alt="1", height="50", width="100">
  <img src="photo2.jpg", alt="2", height="50", width="70">
</body>
